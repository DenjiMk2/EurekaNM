# エウレカNM管理ツール

何番煎じか分からないエウレカNMの沸き状況管理ツール

- NMの名前を見てどこに沸くのかパッと思い出せない  
- 沸いたか沸いてないかだけでなく**大雑把に** (←ここ重要) 経過時間を把握したい  

そんな人向けに作ったつもりです．

画像は[@xiv_bell様のもの](https://twitter.com/xiv_bell/status/974689417947312129)をお借りしました．

## 機能説明

### マップ
- マップ上の円をクリックすると下のリストが更新される
- 円は徐々に大きく赤くなっていき，120分でMAXになる

### モンスターリスト
左の項目から
  - モンスターの適正討伐EL
  - モンスター名
  - 最後に沸いた(クリックした)時間
  - 沸いた時間からの経過時間(秒)
